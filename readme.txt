== Description ==

# CF7 Additional Types

This plugin extends Contact Form 7 by adding addtional input field types.

Implemented Types:

*   Rangeslider
    
    HTML5-based rangeslider (using Ion.RangeSlider jQuery Plugin)

What I am working on for the next version:

*   Datepicker, Timepicker
*   Translations


== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/cf7-additional-types` directory, or install the plugin through the WordPress plugins screen directly.
1. Activate the plugin through the 'Plugins' screen in WordPress
1. Go to Contact &gt; Contact Forms and integrate the new input field types into your Contact Form 7 forms
